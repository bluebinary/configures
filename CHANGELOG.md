# Configures Library Change Log

## [0.9.3] - 2026-02-11
### Added
- Support JSON configuration files with optional single- and multi-line comments and trailing commas.
- Updated `README` file with additional information regarding the `Secrets` class methods, method behaviour, and specification file formats.

## [0.9.2] - 2026-02-10
### Added
- Support for secrets without regular expression or option validation.
- Support for Python 3.14.

## [0.9.1] - 2025-09-17
### Added
- Improved configuration specification file regular expression validation.

## [0.9.0] - 2025-09-04
### Added
- First release of the Configures environment variable access and validation library.
